Checks Wordfence core file to avoid any modification by hacker. Sends alerts about file modifications to email.

== Description ==

Wordfence Security - the most popular wordpress firewall & security scanner, but hackers modify the codes of this plugin and injects own malware code. To avoid all these problem, you can use Security Core Control for Wordfence plugin to control any changes with original plugin.
